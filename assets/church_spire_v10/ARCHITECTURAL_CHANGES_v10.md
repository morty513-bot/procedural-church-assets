# Church Spire v10 — Architectural Change Notes (vs v9)

This v10 model keeps the stylized/low-poly look, but introduces stronger Saint-Émilion-inspired tower/spire cues while remaining original.

## Key architectural upgrades

- **Heavier stone massing at lower tower**
  - Broader base block with stronger corner buttress masses for a more grounded Romanesque/Gothic transition feel.

- **Clear belfry stage**
  - Added a distinct octagonal belfry level instead of a mostly continuous shaft.
  - Belfry includes **8 readable vertical openings** (window proxies) to suggest bell chamber ventilation/open arcades.

- **Steeper two-step roof transition**
  - Introduced a wider lower roof skirt, then a steeper upper spire cone to create a more authentic profile break.
  - This gives a clearer progression from tower to spire rather than one simple cone stack.

- **Ornamental top details**
  - Added a lantern/collar element under finial.
  - Refined top with finial sphere + cross.
  - Added small ring pinnacles/crockets around the transition zone for visual richness.

- **More legible façade openings**
  - Mid-tower slit windows retained and made clearly readable in renders through contrast and emissive treatment.

## Style/render continuity retained

- Preserved approved **world-space Z gradient sky** behavior (stable orientation; no camera-locked drift).
- Kept vivid fantasy palette options while maintaining clearly separated materials:
  - warm stone,
  - dark slate roof,
  - cyan window glow,
  - metallic finial,
  - terracotta ornamental accents.
